## Laravel and Vue js

-   Clone or download this repo and place it into your server.
-   `composer install `
-   `cp .env.example .env `
-   Create database and modify .env with your DB name and Pusher credentials.
-   `php artisan migrate `
-   `php artisan key:generate `
-   `npm install && npm run dev `
-   `php artisan storage:link`
-   `php artisan serve `
